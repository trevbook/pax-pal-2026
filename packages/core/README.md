# @pax-pal/core

Shared types, taxonomy constants, and utilities for the PAX Pal ecosystem.

## Contents

- **Game types** — Staged types for the data pipeline (`RawExhibitor`, `RawDemo`, `HarmonizedGame`, `Game`)
- **Taxonomy constants** — Typed constant arrays for game types, platforms, genres, mechanics, and tags
- **Utilities** — Slug generation and shared helpers

## Usage

```typescript
import { Game, HarmonizedGame, PLATFORMS, VIDEO_GAME_GENRES, toSlug } from "@pax-pal/core";
```
